Reloj CSS Github

Este proyecto recrea un reloj utilizando CSS y vanilla Javascript. Para lograr el funcionamiento de los brazos del reloj, se ejecuta una funcion cada segundo que convierte el segundo actual en grados y usando la propiedad transform de CSS se mueve la posicion del brazo al angulo correspondiente, dando un efecto similar a un reloj real.
